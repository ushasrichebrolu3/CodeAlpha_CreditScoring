# CodeAlpha_CreditScoring
This project is a machine learning model that predicts whether a loan applicant is a credit risk or not. It uses Logistic Regression to classify applicants based on their financial data. The model is trained on historical data and evaluated using accuracy, confusion matrix, ROC curve, and feature importance. 
Credit Scoring Model
📌 Description
